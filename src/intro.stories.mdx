# Ant Design - React Extensions

Extentions for https://ant.design/.

## Installation

### Using Yarn
```
yarn add antd-react-extensions
```

### Using NPM
```
npm install antd-react-extensions --save
```

### Add Styles to your Project

```
import 'antd-react-extensions/dist/index.css';
```
